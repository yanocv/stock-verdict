# StockVerdict

**StockVerdict** helps you make smarter investment decisions by analyzing key financial indicators and providing a clear recommendation on whether a stock is a good buy.

## Features

- Real-time stock data visualization
- Buy/Hold/Sell recommendations based on fundamental metrics
- News analysis and sentiment
- USD/JPY exchange tracking
- English & Japanese language support (i18n ready)

## Tech Stack

- **Frontend**: React + Next.js + TypeScript
- **Styling**: Tailwind CSS
- **Data**: Axios, Puppeteer, Cheerio
- **Testing**: Jest, React Testing Library
- **Deployment**: Vercel (frontend), Render/Heroku (backend)
- **CI/CD**: GitHub Actions

## Run Locally

```bash
# Clone the repo
git clone https://github.com/your-username/stockverdict.git
cd stockverdict

# Install dependencies
npm install

# Start development server
npm run dev
```
